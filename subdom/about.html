Como utilizar o app

Você precisa apenas digitar o valor do produto no segundo campo, e no terceiro campo digitar a quantidade de itens relacionado ao mesmo produto, um valor é obrigatório para todos os campos, e não é permitido valores maiores que 12 caracteres, Adicione os produtos e os valores respectivos, depois clique no botão +(mais). Após feito todos os valores serão calculados automaticamente, você só precisará adicionar os produtos e todo o cálculo será feito e mostrado no VALOR TOTAL.

Para apagar um valor especifico clique em apagar ao lado direito, e para apagar todos os valores de uma vez, clique em ( Limpar lista ).

Para utilizar a lista

Apenas digite o nome do item a ser salvo, e clique em salvar, caso tenha digitado algo errado, clique no ( X ) ao lado do nome para apagar, e para limpar a lista clique no botão abaixo ( Limpar Lista Toda )
